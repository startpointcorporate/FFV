# AI-Policy.md

## 1. What AI Can Do

-   Generate MAUI UI code
-   Suggest BFF controllers
-   Draft unit tests
-   Draft documentation
-   Analyze entropy metrics

------------------------------------------------------------------------

## 2. What AI Cannot Do

-   Merge pull requests
-   Deploy to production
-   Access secret vault
-   Modify governance files
-   Approve release

------------------------------------------------------------------------

## 3. Evidence Requirements

AI contribution must document:

-   Prompt category
-   Code percentage influenced
-   Validation depth
-   Risk classification

------------------------------------------------------------------------

## 4. AI Safety Rules

-   No production secrets in prompts
-   Prompt sanitization required
-   Immutable logging required
-   RBAC enforced
